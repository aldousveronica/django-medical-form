Django medical form by Veronica Aldous 

requires python 3

how to install the requirements: 
pip install -r requirements.txt

how to run: 
python manage.py runserver

core webpages: 
1. http://127.0.0.1:8000/enter
    this webpage takes you to the form to enter a participant 
2. http://127.0.0.1:8000/list 
    this webpage takes you to the list of participants
